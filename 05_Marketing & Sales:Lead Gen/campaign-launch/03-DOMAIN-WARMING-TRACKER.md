## Issue Response Plan

### If Domain Shows Warning Signs:
1. **Immediate:** Reduce volume by 50%
2. **Within 24h:** Investigate cause (content? list quality? sudden spike?)
3. **Within 48h:** Implement fix
4. **Monitor:** Daily checks for 1 week before resuming normal volume

### If Domain Gets Blacklisted:
1. **Immediate:** PAUSE all sending from that domain
2. **Identify:** Which blacklist? (Use MXToolbox)
3. **Request delisting:** Follow blacklist-specific process
4. **Wait:** Until fully delisted (1-7 days typically)
5. **Resume:** Start from Day 1 of warming schedule

### If Multiple Domains Have Issues:
1. **PAUSE ENTIRE CAMPAIGN**
2. **Root cause analysis:** Content? List? Technical issue?
3. **Fix systemically** before resuming any domain
4. **Gradual resume:** One domain at a time
